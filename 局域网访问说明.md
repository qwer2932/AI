# 局域网访问配置说明

## 项目已配置为支持局域网访问

### 启动服务器
```bash
python run.py
```

启动后，控制台会显示：
- 本机访问地址：`http://localhost:5000`
- 局域网访问地址：`http://[您的IP]:5000`

## 防火墙配置

### Windows 系统

1. **打开Windows防火墙设置**
   - 按 `Win + R`，输入 `wf.msc`，回车

2. **添加入站规则**
   - 点击"入站规则" → "新建规则"
   - 选择"端口" → "下一步"
   - 选择"TCP"，输入端口号"5000" → "下一步"
   - 选择"允许连接" → "下一步"
   - 勾选所有网络类型 → "下一步"
   - 输入规则名称"AI视频分析系统" → "完成"

3. **或者临时关闭防火墙（不推荐）**
   - 控制面板 → 系统和安全 → Windows Defender防火墙
   - 点击"启用或关闭Windows Defender防火墙"
   - 临时关闭防火墙进行测试

### Linux 系统

```bash
# Ubuntu/Debian
sudo ufw allow 5000

# CentOS/RHEL
sudo firewall-cmd --permanent --add-port=5000/tcp
sudo firewall-cmd --reload
```

## 网络要求

1. **同一局域网**
   - 服务器和客户端设备必须在同一局域网内
   - 通常连接到同一个路由器/交换机

2. **IP地址获取**
   - 服务器会自动显示本机IP地址
   - 如果IP地址不正确，可以手动查看：
     - Windows: `ipconfig`
     - Linux/Mac: `ifconfig` 或 `ip addr`

## 访问方式

### 本机访问
- 地址：`http://localhost:5000`
- 用途：服务器本机测试

### 局域网访问
- 地址：`http://[服务器IP]:5000`
- 示例：`http://192.168.1.100:5000`
- 用途：其他设备访问

## 故障排除

### 无法访问
1. 检查防火墙设置
2. 确认IP地址正确
3. 确认端口5000未被其他程序占用
4. 检查网络连接

### 端口被占用
```bash
# Windows
netstat -ano | findstr :5000

# Linux/Mac
lsof -i :5000
```

### 修改端口
如果5000端口被占用，可以修改 `run.py` 中的端口号：
```python
port = 8080  # 改为其他端口
```

## 安全建议

1. **仅在内网使用**
   - 不要将服务器暴露到公网
   - 仅在可信任的局域网内使用

2. **定期更新**
   - 保持系统和依赖库更新
   - 定期检查安全补丁

3. **访问控制**
   - 考虑添加用户认证
   - 限制访问IP范围
