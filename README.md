# AI视频追踪分析系统

这是一个基于YOLOv8和DeepSORT的智能视频分析系统，专门用于追踪和分析人员行为，包括增值时间、非增值时间、步行时间和等待时间的统计。

## 功能特点

- **智能视频上传**: 支持拖拽上传，支持MP4、AVI、MOV等格式
- **实时追踪分析**: 使用YOLOv8进行目标检测，DeepSORT进行多目标追踪
- **行为分析**: 自动分析每个追踪目标的四种行为类型
- **数据可视化**: 生成柱状图展示前三个追踪目标的行为分布
- **详细统计**: 提供完整的分析结果表格和统计信息
- **结果视频**: 显示包含追踪框的处理后视频

## 系统要求

- Python 3.8+
- CUDA支持的GPU（推荐）
- 8GB+ RAM

## 安装步骤

1. **克隆项目**
```bash
git clone <项目地址>
cd ai_track
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **启动系统**
```bash
python run.py
```

4. **访问系统**
打开浏览器访问: http://localhost:5000

## 使用方法

### 1. 上传视频
- 点击上传区域选择视频文件
- 或直接拖拽视频文件到上传区域
- 支持格式：MP4, AVI, MOV, MKV, WMV

### 2. 开始分析
- 点击"开始分析"按钮
- 系统会自动进行目标检测和追踪
- 实时显示分析进度

### 3. 查看结果
- **概览统计**: 显示追踪目标数量、分析时长等
- **行为图表**: 柱状图展示前三个ID的行为分布
- **详细表格**: 完整的行为分析数据
- **结果视频**: 包含追踪框的处理后视频

## 行为分析说明

系统会自动分析每个追踪目标的四种行为类型：

- **增值时间**: 目标在特定区域进行有价值活动的时间
- **非增值时间**: 目标进行非生产性活动的时间
- **步行时间**: 目标在移动过程中的时间
- **等待时间**: 目标静止等待的时间

## 技术架构

- **前端**: HTML5 + Tailwind CSS + Chart.js
- **后端**: Flask + Python
- **AI模型**: YOLOv8 + DeepSORT
- **视频处理**: OpenCV

## 注意事项

1. 首次运行时会自动下载YOLOv8模型
2. 如果DeepSORT不可用，系统会自动回退到简单追踪模式
3. 建议使用GPU加速以提高分析速度
4. 大视频文件可能需要较长的处理时间

## 故障排除

### 常见问题

1. **模型加载失败**
   - 检查模型文件路径是否正确
   - 确保有足够的磁盘空间

2. **分析超时**
   - 检查视频文件是否过大
   - 尝试使用较短的视频片段

3. **追踪效果不佳**
   - 确保视频质量良好
   - 调整检测阈值参数
